# Basic Memory

This repository's `docs/` directory is registered with [Basic Memory](https://docs.basicmemory.com/) as project **`claude-code-profiles`**.

- Project name: `claude-code-profiles`
- Notes path: `docs/`

Basic Memory is configured globally as an MCP server for Claude Code (`basic-memory`) and Codex CLI (`mcp_servers.basic-memory`). Agents working on this repository can use Basic Memory tools to search, read, and write structured notes against this project.

See https://docs.basicmemory.com/how-to/project-documentation for project documentation patterns.
